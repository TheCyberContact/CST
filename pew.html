WINDOWS PENETRATION

The following are possible ways to escalate privileges on a Windows box and other bits and bobs

WinPEAS - auto check for vulnerabilities

WinPEAS download from github click here

Check your kernel for known exploits

systeminfo

systeminfo | findstr /B /C:"OS Name" /C:"OS Version"

wmic qfe get Caption,Description,HotFixID,InstalledOn

Check drivers

driverquery

Check for Anti-Virus

WMIC /Node:localhost /Namespace:\\root\SecurityCenter2 Path AntiVirusProduct Get displayName /Format:List | more

Check users and groups

net users

net localgroup

whoami /all

Get-WmiObject -Class Win32_UserAccount

Check clipboard (long shot maybe a password)

powershell -command "Get-Clipboard"

Check privileges

whoami /priv

Check ports

netstat -ano

Check starup and if any are writeable

wmic startup

Check processes

Tasklist /SVC

Check services

wmic service list brief

Check credentials

wmic service list brief

Check Wifi and recover plain text passwords

netsh wlan show profile

netsh wlan show profile PROFILE key=clear

Check SSH credentials

reg query HKEY_CURRENT_USER\Software\OpenSSH\Agent\Keys

Search registry for possible credential locations

reg query "HKLM\SOFTWARE\Microsoft\Windows NT\Currentversion\Winlogon" 2>nul | findstr /i "DefaultDomainName DefaultUserName DefaultPassword AltDefaultDomainName AltDefaultUserName AltDefaultPassword LastUsedUsername"

Search registry for possible entries with "password"

reg query HKLM /f password /t REG_SZ /s
