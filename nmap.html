NMAP COMMANDS

You can download Nmap and find more information here at their official website

Nmap help

nmap -h

Basic scan

nmap 10.0.0.1

My default scan

nmap 10.0.0.1 -Pn -sS -T4 -p- -v -oN scan.txt

Scan an IP range

nmap 10.0.0.1-254

TCP SYN scan

nmap 10.0.0.1 -sS

TCP ACK scan

nmap 10.0.0.1 -sA

UDP scan

nmap 10.0.0.1 -sU

Scan without ping

nmap 10.0.0.1 -Pn

Scan for specific port

nmap 10.0.0.1 -p 22

Scan for port range

nmap 10.0.0.1 -p 1-5555

Scan for all ports

nmap 10.0.0.1 -p-

Scan UDP and TCP specific ports

nmap 10.0.0.1 -p U:53,T:22

Scan for OS

nmap 10.0.0.1 -O

Scan for service and version

nmap 10.0.0.1 -sV

Scan for OS, version, script scanning, and traceroute

nmap 10.0.0.1 -A

Scan timing T0-5 (Slow and accurate to Fast and aggressive)

nmap 10.0.0.1 -T4

Scan with NSE scripts

nmap 10.0.0.1 -sC

Scan via proxy

nmap --proxies http://192.168.1.1:8080 10.0.0.1

Scan output to a file

nmap 10.0.0.1 -oN filename.txt

Scan output to muiltiple formats (txt,xml,grepable)

nmap 10.0.0.1 -oA filename.file

Scan with verbose output (more v's for more output)

nmap 10.0.0.1 -v

Scan shows only open ports

nmap 10.0.0.1 --open

Scan with ping only

nmap 10.0.0.1 -sn
